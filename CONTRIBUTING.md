- **Software** (all code including firmware, host software, and web interfaces): MIT License + Commons Clause
- **Documentation** (guides, images, videos, written content): CC BY-NC-SA 4.0

This applies to all versions of the basically Sorter project (V1, V2, and beyond).

## What This Means

- You retain copyright to your contributions
- Your contributions can be used for non-commercial purposes by anyone
- Commercial use requires a separate license from Basically Technologies LLC
- Your contributions become part of the source-available basically Sorter project

## Non-Commercial vs Commercial Use

**Non-commercial use (free):**
- Personal hobby projects
- Educational purposes
- Research and development

**Commercial use (requires license):**
- Manufacturing and selling Sorters
- Using Sorters in business operations
- Offering sorting as a commercial service

For commercial licensing: contact@basically.website

## Types of Contributions

- Pull requests (code, CAD files, documentation)
- Bug reports and feature requests
- Design improvements and suggestions
- Documentation improvements
- Community support and discussions
